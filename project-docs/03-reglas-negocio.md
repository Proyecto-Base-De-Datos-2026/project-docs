
Reglas de negocio

•	Una mascota debe tener un dueño obligatorio 

•	Un dueño puede tener varias mascotas 

•	Una cita debe estar asociada a una mascota y un veterinario 

•	No pueden existir dos citas al mismo tiempo con el mismo veterinario 

•	Cada cita genera una historia clínica 

•	El diagnóstico en la historia clínica es obligatorio 

•	Un tratamiento pertenece a una historia clínica 

•	Un tratamiento puede incluir varios medicamentos 

•	Cada cita debe tener un pago 

•	El monto del pago debe ser mayor a 0
