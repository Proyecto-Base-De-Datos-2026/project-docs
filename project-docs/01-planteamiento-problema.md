Descripción del proyecto
El presente proyecto tiene como objetivo el desarrollo de una base de datos para la gestión de una clínica veterinaria. El sistema permitirá administrar la información relacionada con dueños, mascotas, citas médicas, historias clínicas, tratamientos, medicamentos y pagos, facilitando el control y organización de los procesos internos de la clínica.

Planteamiento del problema
Una clínica veterinaria necesita gestionar sus operaciones diarias: registrar mascotas y sus dueños, agendar citas médicas, llevar historias clínicas, controlar tratamientos y medicamentos, y registrar pagos.
Sin un sistema centralizado, la clínica pierde información médica importante, genera errores en la programación de citas y no puede hacer un seguimiento adecuado de los tratamientos, afectando la calidad del servicio.

Objetivo general
Diseñar e implementar una base de datos que permita gestionar de manera eficiente la información de una clínica veterinaria, garantizando integridad, organización y fácil acceso a los datos.
